# TABLO 🎨🧠
**Real-time Socratic AI Whiteboard for Collaborative Learning**

Tablo is an interactive educational platform where learning isn't a chat—it's a conversation. Built for **UN SDG 4 (Quality Education)**, Tablo uses a multimodal AI "Co-Thinker" that sees your sketches, hears your questions, and nudges you toward mastery through the Socratic method.

## 🚀 The Vision
Most AI tutors just give you the answer. Tablo picks up a digital pen and draws you a hint. It's designed to reduce the "friction of understanding" by combining a shared infinite canvas with a live, low-latency multimodal brain.

## 🛠️ The 2026 Tech Stack
- **Frontend:** Next.js 16 (App Router) & tldraw (Vector Canvas)
- **Backend:** FastAPI (Python) & LangGraph (Agentic Logic)
- **Intelligence:** Gemini 3.1 Flash Live (Multimodal WebSocket Stream)
- **Tools:** MCP (Model Context Protocol) for Scientific Computing
- **Memory:** Multimodal RAG for PDF/Textbook Grounding

- **Week 2:** Socratic Interaction & Logic Drawing
- **Week 3:** RAG & Scientific Tool Integration
